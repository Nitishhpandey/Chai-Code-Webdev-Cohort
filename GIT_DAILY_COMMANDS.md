# Daily Git Workflow — Chai Code Cohort

## After every lecture (run these 3 commands)

```bash
git add .
git commit -m "Module-03 Lec-5: objects methods optional chaining"
git push origin main
```

## Commit message format
```
Module-XX Lec-X: [what you learned today]

Examples:
Module-02 Lec-4: CSS flexbox layout practice
Module-03 Lec-7: destructuring arrays and objects
Module-09 Lec-3: first HTTP server with Node.js
Module-11 Lec-4: useState and lifting state
```

## Rule: commit SAME DAY as the lecture
Never let more than 1 day pass without a commit.
