## 📝 Notes File Format

Every sub-topic folder contains:
- `notes.md` — key concepts, definitions, code snippets
- `index.html` / `script.js` / `app.js` — hands-on code written during lecture
- `practice/` — extra practice problems done after lecture

**notes.md template:**
```markdown
## [Topic Name]
**Date:** DD-MM-YYYY
**Lecture:** Module X - Lec X

### Key Concepts
- concept 1
- concept 2

### Code I Wrote
[paste key snippet here]

### What I Didn't Understand
- note it here, revisit next day

### Resources
- [link if any]
```
